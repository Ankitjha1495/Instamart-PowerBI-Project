
# Instamart Sales Analytics Dashboard

## Project Overview

Developed an interactive Power BI dashboard to analyze Instamart sales performance, customer purchasing behavior, and business KPIs. The dashboard provides actionable insights into revenue trends, category performance, order patterns, and operational efficiency to support data-driven decision-making.

## Tools & Technologies

* Power BI
* Power Query
* DAX
* Microsoft Excel
* Data Visualization

## Key Features

* Interactive KPI dashboards
* Sales and revenue trend analysis
* Category-wise performance tracking
* Customer order behavior analysis
* Dynamic filtering and drill-through reporting
* Data cleaning and transformation using Power Query

## Business Insights

* Identified top-performing product categories and sales trends
* Analyzed customer purchasing patterns and order frequency
* Improved visibility into operational and sales performance metrics
* Enabled better business decisions through real-time visual analytics

## Skills Demonstrated

* Data Analysis
* Dashboard Development
* KPI Reporting
* Business Intelligence
* Data Visualization
* DAX Calculations
* ETL & Data Transformation

## Outcome

This project demonstrates the ability to transform raw business data into meaningful insights using Power BI and analytical reporting techniques.
